# This Repository Has Moved

This repository contained the empirical experiments for an earlier
preprint on neutrosophic statistical methods. The work has been moved
and expanded to a new repository:

➡️ **https://github.com/mleyvaz/layered-neutrosophic-symmetry**

The new repository contains:

- All original experiments from the preprint version
- Additional empirical validation
- The `thirdanswer` Python library

Please update your bookmarks. This repository is preserved for citation
continuity but is no longer maintained.

---

For the published version of this work, see the Special Issue
*Symmetry in Uncertainty and Intelligent Decision-Making*
in the journal **Symmetry** (MDPI).
